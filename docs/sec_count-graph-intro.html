Section 5.1 The Chromatic Function

¶

And now for the grand finale! Let's use all the tools we have developed so far to try to understand graph colorings better.

Recall that a proper coloring of a graph is an assignment of colors to the vertices of the graph such that adjacent vertices must be colored differently. The interesting question about such colorings is: what is the smallest number of colors you can use to properly color the graph? This smallest number is called the chromatic number of the graph (written \(\chi(G)\)).

We have seen that in general, there is no limit to how large the chromatic number of a graph can be. In particular, the complete graph \(K_n\) (the graph with \(n\) vertices all of which are adjacent) has chromatic number \(\chi(K_n) = n\text{.}\) However, a very early question in graph theory was whether every planar graph has chromatic number at most 4. This is called the Four Color Theorem. This theorem seems like it should be fairly easy to prove, but it took mathematicians a long time to prove it, and even today, the best proof we have requires a computer to check hundreds of cases. Given the difficulty of this theorem, graph theorists have spent a lot of effort to understand chromatic number in many different ways.

The topic we are about to consider is one such attempt at understanding the chromatic number.

Subsection Counting Colorings

¶

It is often helpful in mathematics to make our problems harder.

Suppose we wanted to decide whether a particular graph \(G\) had chromatic number 4. This would mean that there is a proper coloring using four colors, but no proper coloring that uses three. Instead of asking whether there is a 4-coloring and whether there is a 3-coloring, let's ask how many 4-colorings and how many 3-colorings there are. In fact, let's ask how many \(k\)-colorings there are, for any natural number \(k\text{.}\)

We have turned our question about graphs into a question about sequences. We can let \((g_n)_{n \ge 1}\) be the sequence in which \(g_n\) is the number of \(n\)-colorings of the graph \(G\text{.}\) Perhaps we can understand this sequence for a particular graph. The first \(n\) such that \(g_n \gt 0\) will be the chromatic number of the graph. Maybe we can even understand something about all these sequences for all planar graphs! 1 

Don't get too excited though, it turns out that so far, this approach has not resulted in a nicer proof of the four color theorem. Maybe some day.

Since the number of \(k\)-colorings of a graph \(G\) depends both on \(k\) and \(G\text{,}\) we will adopt the following notation.

Definition 5.1.1.

For any graph \(G\text{,}\) the chromatic function of \(G\text{,}\) denoted \(P(G,k)\text{,}\) gives the number of proper colorings of \(G\) that use up to \(k\) colors.

Once \(G\) is fixed, \(P(G,k)\) is a function of \(k\text{.}\) Later we will discover that this function is always a polynomial, and at that point we will call \(P(G,k)\) the chromatic polynomial of a graph.

Note: it is perfectly reasonable to have \(k\) be larger than the number of vertices. Think of \(k\) as telling you how many colors you have in your crayon box. Each vertex must be one of those colors (observing the rule that adjacent vertices must be colored differently), but you are not required to use all of the colors in the box.

To become familiar with the chromatic function, try the following activity.

Worksheet The Chromatic Function

¶

For any graph \(G\text{,}\) let \(P(G,k)\) denote the number of proper colorings of \(G\) that use up to \(k\) colors (think of having \(k\) colors in your crayon box you can use to color the vertices; you don't have to use all of them). For a fixed graph \(G\text{,}\) \(P(G,k)\) is a function of \(k\text{.}\)

1.

For each graph \(G\) below, find \(P(G,2)\text{,}\) \(P(G,3)\text{,}\) and \(P(G,4)\text{.}\) Note, these graphs are labeled, so coloring vertex \(a\) red and \(b\) blue is a different coloring than coloring \(a\) blue and \(b\) red.

 

 

Solution

First let's find the chromatic functions for the paths on the left. The top graph is \(P_1\text{,}\) and we have \(P(P_1, 2) = 2\) (since we can pick any of the two colors for \(a\text{,}\) and the other for \(b\)), \(P(P_1, 3) = 6\text{,}\) and \(P(P_1, 4) = 12\) (there are 4 choices for \(a\) and 3 choices for \(b\)).

For the second path, \(P_2\text{,}\) we have \(P(P_2, 2) = 2\text{,}\) \(P(P_2,3) = 12\text{,}\) and \(P(P_2, 4) = 36\text{,}\) since there are 4 colors to choose from for \(a\text{,}\) then three for \(b\) and three for \(c\text{.}\)

For \(P_3\text{,}\) we have \(P(P_3, 2) = 2\text{,}\) \(P(P_3, 3) = 24\text{,}\) and \(P(P_3,4) = 108\text{.}\)

For \(C_3 = K_3\) (the triangle on the top right), we have \(P(K_3, 2) = 0\) (there are no proper colorings of \(K_3\) with 2 colors), \(P(K_3, 3) = 6\text{,}\) and \(P(K_3, 4) = 24\) (since there are 4 colors you can pick for \(a\text{,}\) 3 colors to choose from for \(b\) and 2 colors you can pick for \(c\)).

For \(K_4\text{,}\) we have \(P(K_4,2) = P(K_4,3) = 0\) and \(P(K_4,4) = 24\text{.}\)

2.

Generalize: Find \(P(P_3,k)\) for any \(k\) (recall \(P_3\) is the path with 4 vertices and 3 edges) and \(P(C_3, k)\) for any \(k\) (where \(C_3\) is a 3-edge cycle, the same as \(K_3\)). That is, find a closed formula for each of these.

Solution

Consider the labeled \(P_3\) shown above. We have \(P(P_3, k) = k(k-1)^3\text{.}\) This is because there are \(k\) choices for \(a\text{.}\) Then there are \(k-1\) choices for \(b\text{.}\) The next vertex in the path must be colored different from \(b\text{,}\) but could be colored the same as \(a\text{,}\) so there are still \(k-1\) choices for its color. Similarly, there are \(k-1\) choices for the color of \(d\text{.}\)

On the other hand, \(P(C_3, k) = k(k-1)(k-2)\) (which is the same as \(P(k, 3)\text{,}\) where this \(P\) represents permutations). There are \(k\) choices for \(a\text{,}\) then only \(k-1\) choices for \(b\) and then \(k-2\) choices for the last vertex, since it cannot be the same as either of the other two.

3.

Use your answers to the previous problem to find \(P(C_4, k)\text{.}\) Hint: If there wasn't an edge between \(a\) and \(d\text{,}\) then we this would be coloring \(P_3\text{.}\) The colorings that we don't want to allow are those in which \(a\) and \(d\) are the same: how many of those are there?

 

Solution

The answer is not \(k(k-1)^3 = P(P_3, k)\text{,}\) but only because some of the colorings of \(P_3\) have \(a\) and \(d\) colored the same. Let's count how many proper colorings of \(P_3\) like that there are. We have \(k\) colors to choose for \(a\) (and this color is the same as \(d\)). Then there are \(k-1\) choices for the color of \(b\text{.}\) What about \(c\text{?}\) Well, we are only considering proper colorings of \(P_3\text{,}\) so \(c\) cannot be the same color as \(b\) or as \(d\) (and we know that \(d\) is the same color as \(a\)). So there are only \(k-2\) colors to choose from for \(c\text{.}\)

Putting this all together we see that \(P(C_4, k) = k(k-1)^3 - k(k-1)(k-2)\text{.}\) Note that the \(k(k-1)(k-2)\) bad colorings we remove are equal in number to the proper colorings of \(C_3\text{.}\) This makes sense: by insisting that \(a\) and \(d\) are colored the same, it is like we are identifying those two vertices. If you slide \(a\) and \(d\) together (removing the redundant edge between them) you get \(C_3\text{.}\)

Subsection Chromatic Functions for Nice Graphs

¶

As the activity above suggests, finding the chromatic function is reasonable for some nice graphs. In all of the cases below, we are always considering labeled graphs: which vertex gets which color matters.

We begin by considering paths.

Theorem 5.1.2.

The chromatic function of \(P_n\text{,}\) the \(n\)-edge path, is \(P(P_n, k) = k(k-1)^n\text{.}\)

Proof.

The graph \(P_n\) contains \(n+1\) vertices, call them \(v_0, v_1, \ldots, v_n\text{,}\) where \(v_0\) and \(v_n\) are the “end” vertices (the only two with degree 1).

There are \(k\) choices for the color of \(v_0\text{.}\) Then since \(v_1\) is adjacent to \(v_0\text{,}\) there are only \(k-1\) choices for \(v_1\text{.}\) Next \(v_2\) is adjacent to \(v_1\text{,}\) so there are \(k-1\) choices for its color: any color except the one we picked for \(v_1\text{.}\) In fact, \(v_i\) can be colored any of the m \(k-1\) colors we did not use for \(v_{i-1}\) (for \(1 \le i \le n\)).

By the product principle, we see that \(P(P_n, k) = k(k-1)^n\text{.}\)

Next, complete graphs.

Theorem 5.1.3.

The chromatic function of the complete graph \(K_n\) is \(P(K_n, k) = \frac{k!}{(k-n)!}\) for \(k \ge n\) and 0 otherwise.

Proof.

If \(k \lt n\text{,}\) then there are no proper \(k\)-colorings of \(K_n\text{,}\) since the chromatic number of \(K_n\) is \(n\text{.}\) So assume that \(k \ge n\text{.}\)

Now there are \(k\) choices for the color of the first vertex. There are only \(k-1\) choices for the color of the second vertex. The vertex after that cannot be either of the first two colors, so there are \(k-2\) choices for its color. And so on. So we have \(P(K_n, k) = k (k-1) (k-2) \cdots (k-n+1)\) (the product contains \(n\) terms), which is \(\frac{k!}{(k-n)!}\text{.}\)

Since all the vertices must have different colors in a complete graph, we can think of a coloring as an injective function from the set of vertices to the set of colors. So it is not surprising then that the number of proper colorings is a \(k\)-permutation of the \(n\) colors. This does, however, lead to the strange and perhaps notationally confusing identity,

\begin{equation*} P(K_n,k) = P(k,n)\text{.} \end{equation*}

What about other classes of graphs? There are some that are not too difficult (you are asked to describe the chromatic function for all trees in the exercises). However, finding the chromatic function for many graphs can be quite challenging. In the next few sections we will consider some techniques to overcome some of this challenge. We conclude with one example of a small graph with a complicated chromatic function computation.

Example 5.1.4.

Find the chromatic function \(P(G,k)\) for the graph \(G\) below.

Solution

There are \(k\) choices for the color of \(v_1\text{,}\) and then \(k-1\) choices for \(v_2\text{,}\) \(k-2\) choices for \(v_3\text{.}\) So far so good. Now \(v_4\) can be any color except the color of \(v_3\text{,}\) so there are \(k-1\) choices for it. What about \(v_5\text{?}\) The problem is that it could be that \(v_4\) and \(v_1\) are the same color, which would mean that \(v_5\) could be any of \(k-1\) colors. But if \(v_1\) and \(v_4\) are different, then \(v_5\) can only be \(k-2\) colors.

At this point, our best bet is to break this into two cases. First, if \(v_1\) and \(v_4\) are different colors, then we will have

\begin{equation*} k(k-1)(k-2)(k-2)(k-2) \end{equation*}

colorings. If \(v_1\) and \(v_4\) are the same color, then we have

\begin{equation*} k(k-1)(k-2)1(k-1) \end{equation*}

colorings. So all together, we see

\begin{equation*} P(G,k) = k(k-1)(k-2)(k-2)(k-2) + k(k-1)(k-2)1(k-1)\text{.} \end{equation*}

In future sections we will see other ways to compute this function.
