There are \(k\) choices for the color of \(v_0\text{.}\) Then since \(v_1\) is adjacent to \(v_0\text{,}\) there are only \(k-1\) choices for \(v_1\text{.}\) Next \(v_2\) is adjacent to \(v_1\text{,}\) so there are \(k-1\) choices for its color: any color except the one we picked for \(v_1\text{.}\) In fact, \(v_i\) can be colored any of the m \(k-1\) colors we did not use for \(v_{i-1}\) (for \(1 \le i \le n\)).
