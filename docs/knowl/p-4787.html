Suppose we wanted to decide whether a particular graph \(G\) had chromatic number 4. This would mean that there is a proper coloring using four colors, but no proper coloring that uses three. Instead of asking whether there is a 4-coloring and whether there is a 3-coloring, let's ask how many 4-colorings and how many 3-colorings there are. In fact, let's ask how many \(k\)-colorings there are, for any natural number \(k\text{.}\)
