Each vertex in \(K_n\) is adjacent to \(n-1\) other vertices, so the degree of each vertex is \(n-1\text{.}\) The handshake lemma tells us how to find the number of edges: the sum of the degrees is \(n(n-1)\text{,}\) since we count \(n-1\) edges \(n\) times (once for each vertex). This is twice the number of edges (since each edge is incident to 2 vertices, so we counted every edge exactly twice). Thus there are \(n(n-1)/2\) edges in \(K_n\text{.}\)
