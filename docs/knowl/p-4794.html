For any graph \(G\text{,}\) let \(P(G,k)\) denote the number of proper colorings of \(G\) that use up to \(k\) colors (think of having \(k\) colors in your crayon box you can use to color the vertices; you don't have to use all of them). For a fixed graph \(G\text{,}\) \(P(G,k)\) is a function of \(k\text{.}\)
