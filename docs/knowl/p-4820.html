Since all the vertices must have different colors in a complete graph, we can think of a coloring as an injective function from the set of vertices to the set of colors. So it is not surprising then that the number of proper colorings is a \(k\)-permutation of the \(n\) colors. This does, however, lead to the strange and perhaps notationally confusing identity,

\begin{equation*} P(K_n,k) = P(k,n)\text{.} \end{equation*}
