  1. \(f = \twoline{1 \amp 2 \amp 3 \amp 4 \amp 5}{1 \amp 2 \amp 1 \amp 2 \amp 1}\text{.}\)

    • Injective

    • Surjective

    • Bijective

    • Neither

  2. \(f = \twoline{1 \amp 2 \amp 3 \amp 4 \amp 5}{1 \amp 2 \amp 3 \amp 1 \amp 2}\text{.}\)

    • Injective

    • Surjective

    • Bijective

    • Neither

  3. \(f(x) = \begin{cases} x \amp \text{ if } x \le 3 \\ x-3 \amp \text{ if } x \gt 3\end{cases}\text{.}\)

    • Injective

    • Surjective

    • Bijective

    • Neither
