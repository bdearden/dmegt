Now there are \(k\) choices for the color of the first vertex. There are only \(k-1\) choices for the color of the second vertex. The vertex after that cannot be either of the first two colors, so there are \(k-2\) choices for its color. And so on. So we have \(P(K_n, k) = k (k-1) (k-2) \cdots (k-n+1)\) (the product contains \(n\) terms), which is \(\frac{k!}{(k-n)!}\text{.}\)
