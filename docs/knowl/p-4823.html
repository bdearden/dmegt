There are \(k\) choices for the color of \(v_1\text{,}\) and then \(k-1\) choices for \(v_2\text{,}\) \(k-2\) choices for \(v_3\text{.}\) So far so good. Now \(v_4\) can be any color except the color of \(v_3\text{,}\) so there are \(k-1\) choices for it. What about \(v_5\text{?}\) The problem is that it could be that \(v_4\) and \(v_1\) are the same color, which would mean that \(v_5\) could be any of \(k-1\) colors. But if \(v_1\) and \(v_4\) are different, then \(v_5\) can only be \(k-2\) colors.
