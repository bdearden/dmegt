We have turned our question about graphs into a question about sequences. We can let \((g_n)_{n \ge 1}\) be the sequence in which \(g_n\) is the number of \(n\)-colorings of the graph \(G\text{.}\) Perhaps we can understand this sequence for a particular graph. The first \(n\) such that \(g_n \gt 0\) will be the chromatic number of the graph. Maybe we can even understand something about all these sequences for all planar graphs! 1
