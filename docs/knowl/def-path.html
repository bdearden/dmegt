Definition 1.5.6.

A path in a graph \(G\) is a subgraph of \(G\) isomorphic to \(P_n\) for some \(n\) (we say that \(n\) is the length of the path).

We say that vertices \(u\) and \(v\) are on a path, or that there is a path from \(u\) to \(v\), provided the subgraph isomorphic to \(P_n\) includes \(u\) and \(v\text{.}\)
